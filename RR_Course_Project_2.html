Synopsis

Weather events have both an economic and population health impact on communities across the county. The US National Oceanic and Atmospheric Administration’s (NOAA) storm database allows us to analyze the impact of these events. The data base tracks characteristics of these events including, where and when they occur, estimates of fatalities, injuries, and damage to property and crops. Using this data I analyzed the economic and health impact of different event types. I am working to address the following questions:

  • Across the US, which types of weather events are most harmful with respect to population health?
  • Across the US, which types of events have the greatest economic consequences?

3. Summarize Population Health Info

In order to address the impact of weather events on population health, I first needed to look a that the two health variables in the data set - fatalities and injuries.

3.1 Calculate fatalaties and injuries

storm_health <- stormdata2 %>% select(evtype, fatalities, injuries)
storm_fatalities <- stormdata2 %>% select(evtype, fatalities) %>% group_by(evtype) %>%
       summarise(total_fatalities = sum(fatalities)) %>% arrange(-total_fatalities)
storm_injuries <- stormdata2 %>% select(evtype, injuries) %>% group_by(evtype) %>%
       summarise(total_injuries = sum(injuries)) %>% arrange(-total_injuries)

3.2 Combine injuries and fatalaties

After looking at fatalities and injuries separately, I chose to combine the two variables into a single variable that showed the total number of casualties. A casualty is a person killed or injured in a war or accident, so I used the mutate() object from the dplyr package to add a new variable called total_health to the data frame. Finally, I summarized the storm health data by the event type and number of casualties.

storm_health <- mutate(storm_health,total_health = fatalities + injuries)
storm_health_total <- storm_health %>% group_by(evtype) %>% summarize(total_health_evtype
       = sum(total_health))%>% arrange(-total_health_evtype)

4. Summarize Economic Info

The processing of economic impact data was more complex because it required more data transformation than the health data. The propdmg and cropdmg variable provide the scale, while the propdmgexp and cropdmgexp variables indicate the parameters. Based on the documentation for the data set, the parameters can be defined as:

  • H, h -> hundreds = x100
  • K, K -> kilos = x1,000
  • M, m -> millions = x1,000,000
  • B,b -> billions = x1,000,000,000
  • (+) -> x1
  • (-) -> x0
  • (?) -> x0
  • blank -> x0

4.1 Convert symbols to numbers

storm_damage <- stormdata2 %>% select(evtype, propdmg,propdmgexp,cropdmg,cropdmgexp)
symbol <- sort(unique(as.character(storm_damage$propdmgexp)))
multiplier <- c(0,0,0,1,10,10,10,10,10,10,10,10,10,10^9,10^2,10^2,10^3,10^6,10^6)
convert_multiplier <- data.frame(symbol, multiplier)
storm_damage$prop_multiplier <- convert_multiplier$multiplier[match(storm_damage$propdmgexp, 
       convert_multiplier$symbol)]
storm_damage$crop_multiplier <- convert_multiplier$multiplier[match(storm_damage$cropdmgexp, 
       convert_multiplier$symbol)]

4.2 Combine and Summarize Economic Impact

With the data transformed, I combined the crop damage and property damage variables into a single variable called storm_damage, again using the mutate() object from the dplyr package. Then I summarized the data based on the event type.

storm_damage <- storm_damage %>% mutate(propdmg = propdmg*prop_multiplier) %>%
       mutate(cropdmg = cropdmg*crop_multiplier) %>% mutate(total_damage = propdmg+cropdmg)
storm_damage_total <- storm_damage %>% group_by(evtype) %>% summarize(total_damage_evtype
       = sum(total_damage))%>% arrange(-total_damage_evtype)

Results

After processing the data, I was able to create graphics that help address to the two questions that began the report.

1. Population Health Impact

My original question was, “Across the US, which types of events are most harmful with respect to population health?” Based on the information contained in the graph below, the answer is tornadoes, which accounts for 96979 injuries and fatalities. This is far above excessive heat.

ggplot(data = storm_health_total[1:10,], aes(x = reorder(evtype, total_health_evtype), 
       y = total_health_evtype)) + geom_bar(stat = "identity") + theme(axis.text.x =  
       element_text(angle=90, vjust=0.5, hjust=1)) + 
       ggtitle("Top 10 Events by Total Number of Casualties") + 
       labs(x = "Event Type", y = "Total Casualties")

2. Economic Impact

My original question was, “Across the US, which types of events have the greatest economic consequences?” Based on the information contained in the graph below, the answer is floods, which accounts for $150,319,678,250 in damage. This is far above hurricane/typhoon at number two.

ggplot(data = storm_damage_total[1:10,], aes(x = reorder(evtype, total_damage_evtype),
